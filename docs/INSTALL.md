# Installation

Project Phoenix is designed to be lightweight and simple to install.

The core version uses standard Linux tools:

- Bash
- rsync
- OpenSSH client
- find
- du

Docker is optional but recommended if you want Docker discovery and inventory features.

## 1. Clone or Copy Project Phoenix

Clone the repository or copy the Project Phoenix folder onto your server.

Example install location:

```text
/opt/project-phoenix/
