# Project Phoenix Branding

## Official Logo

The current official Project Phoenix logo is:

```text
assets/branding/phoenix-logo-v1.png
```

A starter scalable vector version is provided as:

```text
assets/branding/phoenix-logo-v1.svg
```

The PNG is the current visual source of truth. The SVG is an editable vector starting point that can be refined over time.

## Tagline

```text
Rise. Recover. Restore.
```

## Brand Direction

Project Phoenix branding should feel:

- simple
- clean
- recognisable
- lightweight
- recovery-focused
- warm and energetic

## Logo Notes

The phoenix symbol represents recovery after failure. The orange-to-red gradient suggests flame, renewal, and resilience.

The original logo should not be overwritten. Future revisions should use versioned filenames such as:

```text
phoenix-logo-v2.png
phoenix-logo-v2.svg
```

## Suggested Uses

Use the phoenix logo for:

- GitHub repository avatar
- README header
- documentation
- future HTML reports
- future web dashboard
- release artwork

## Do Not

- Do not overwrite `phoenix-logo-v1.png`.
- Do not stretch the logo out of proportion.
- Do not add unnecessary text inside the symbol.
- Do not commit temporary logo experiments unless they are intentionally kept.
